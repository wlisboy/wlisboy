使用方式
```
# 参数
 -v, --version  指定版本号与 -l 或 -b 配合使用,
                 如 -v 6.42 Build 64 -l 或 -v 6.42 Build 64 -b,
                 可前往 https://www.internetdownloadmanager.com/news.html 获取版本号
  -h, --help     显示帮助信息
  -i, --info     查询最新版本号
  -l, --list     列出默认下载地址
  -b, --best     列出加速下载地址
```
建议搭配 [IDM-Activation-Script-Chinese](https://github.com/tytsxai/IDM-Activation-Script-Chinese/releases/latest) 使用
