使用方式
```
# 建议搭配（https://github.com/tytsxai/IDM-Activation-Script-Chinese/releases/latest）使用
py idm.py --version xxx --list 获取指定版本号默认下载地址
py idm.py --version xxx --best 获取指定版本号加速下载地址
py idm.py --help    显示帮助信息
py idm.py --info    查询最新版本号
py idm.py --list    列出最新默认下载地址
py idm.py --best    列出最新加速下载地址
```
