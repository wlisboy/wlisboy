## 运行参数
| 参数 | 必选 | 说明 |
|------|------|------|
| `-4` /`--ipv4` | 二选一 | 查询 IPv4 Prefix |
| `-6` / `--ipv6` | 二选一 | 查询 IPv6 Prefix |
| `-a` / `--asn` | ✓ | 目标 ASN（如 `906`） |
| `-t` / `--time` | ✓ | 目标日期（如 `20260722`） |
| `-o` / `--output` | ✓ | 输出 CSV 文件名（如 `result.csv`） |
