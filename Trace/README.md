# Trace

**批量去程精品网线路判定工具**

Trace 是一款专批量去程精品网线路判定工具。它通过对目标 IP/域名执行 `tracert`，实时匹配路由路径中的特征 ASN，快速判定目标是否途经 **CN2 GIA/GT、CN2、CUII（A网）、CMIN2、CMI** 等精品线路。

```
┌─────────────────────────────────────────────────────┐
│  目标列表          Go 调度器          Python 判定    │
│  ip.txt ──→  main.exe (并发 tracert) ──→  CSV 报表   │
│   (输入)         (探测+匹配)           (.csv)        │
└─────────────────────────────────────────────────────┘
```

## 功能特性

- **批量并发探测** — Go 并发 Worker Pool，毫秒级完成大批量目标扫描
- **逐跳实时可见** — 流式输出每个目标的探测进度，实时观察路由路径
- **命中即停** — 一旦识别到特征 ASN，立即终止 tracert，提升效率
- **精品线路判定** — 自动匹配以下特征 ASN 并给出线路类型：

| 特征 ASN | 运营商 | 线路类型 |
|----------|--------|----------|
| AS4809 | 中国电信 | CN2 GIA/GT |
| AS4812 | 中国电信 | CN2 |
| AS9929 | 中国联通 | CUII（A网） |
| AS58807 | 中国移动 | CMIN2 |
| AS58453 | 中国移动 | CMI |

- **ASN 前缀离线库** — 从 RIPEStat 拉取最新前缀，本地缓存，离线可用
- **CSV 输出** — 含 BOM 头，Excel 可直接打开无乱码

## 快速开始

### 运行

```bash
# 准备目标列表（每行一个 IP 或域名）
echo "1.1.1.1" > ip.txt
echo "8.8.8.8" >> ip.txt
echo "example.com" >> ip.txt

# 批量测试，输出 CSV 报表
trace.exe -f ip.txt -o result.csv

# 指定并发数和探测参数
trace.exe -f ip.txt -o result.csv -t 16 -ho 30 -ht 1500
```

启动时会提示是否更新 `asn_prefixes.json`（首次运行选 `y`），之后即可离线重复使用。

### 命令行参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `-f` / `--file` || 导入测试 IP/域名 的列表文件（`.txt`，每行一个） |
| `-o` / `--output` || 导出测试结果的 CSV 文件路径 |
| `-t` / `--threads` | 8 | 并发探测线程数 |
| `-ho` / `--hops` | 30 | tracert 最大跳数 |
| `-ht` / `--hop-timeout` | 1500 | 单跳超时毫秒数 |
