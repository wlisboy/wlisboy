## 运行参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `-f` / `--file` | — | 目标列表文件（每行一个 IPv4/IPv6/IP:port/CIDR） |
| `-o` / `--output` | — | CSV 结果输出路径 |
| `-t` / `--threads` | 8 | 并发探测线程数 |
| `-ho` / `--hops` | 30 | tracert 最大跳数 |
| `-ht` / `--hop-timeout` | 1500 | 单跳超时（毫秒） |
| `-url` / `--speed-url` | 测速地址 | HTTP/HTTPS 测速地址 |

## 线路识别
