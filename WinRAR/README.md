使用方式
```
# 参数
-i, --info            输出最新 Version 版本号
-l, --latest          输出 RARLAB 官网中文版下载地址
-b, --best            输出商业中文版下载地址
-v VERSION, --version VERSION
                        指定历史版本（如 7.20），需与 -l 或 -b 搭配使用，>=7.2

# 示例  
py winrar.py -v 7.20 -l
py winrar.py -v 7.20 -b
```
