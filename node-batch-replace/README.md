# Node Batch Replace — 节点批量替换工具

一款浏览器端的代理节点批量替换工具，目前支持 **vless / trojan / Shadowsocks（ss）** 协议的地址替换。

## 实现功能

- **节点替换**：将原始节点链接中的 IP/域名批量替换为目标地址，支持 `vless://`、`trojan://`、`ss://` 三种协议
- **地址格式支持**：
  - IPv4 / IPv6
  - 域名
  - CIDR 段自动展开（`1.2.3.0/24` → 256 个 IP，最大 65536 个）
- **目标地址携带备注**：通过 `#别名` 语法为每个目标指定节点别名（`1.2.3.4#别名`）
- **端口保留与替换**：目标地址可指定端口（`1.2.3.4:8443`），未指定则保持原端口
- **获取方式**：支持**顺序**取前 N 个 或 **随机**取 N 个
- **提取地址**：**点击开关**自动从原始节点中提取 IP/域名填入目标列表

## 使用方法

1. 在**原始节点**文本框粘贴需要处理的节点链接（每行一个）
2. 在 **IP/域名** 文本框输入目标地址列表（支持 `IPv4`、`IPv6`、`域名`、`CIDR`，每行一个，可选 `#备注`）
3. 选择获取方式（顺序/随机）和获取数量
4. 点击「开始处理」生成替换后的节点列表
